Инструкция запуска на windows: 
1 способ: Можно запустить через среду разработки, например PyCharm, выгрузив requirements.txt и скомпилировать.
2 способ(более универсальный):в папке с проектом сделать bush и ввести сначала команду  "pip3 install -r requirements.txt", затем "python main.py".
После того как программа запуститься, вас попросят ввести url сайта. После чего происходит обработка изображений с сайта.
Обработка занимает некоторое время, из-за того что для вычисления разрешения картинки, нужно скачивать картинку, после чего брать значения и удалять.
(Реализовать по-другому для более быстрой работы я не понял как сделать).
После того как все картинки были обработаны, в папке в файле output.txt выводится список в виде: url картинки, разрешение картинки(x, y), размер картинки в KB.
